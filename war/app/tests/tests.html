Sami Adranly In older IEs, page coordinates can be calculated by adding document scroll to clientX/clientY. If the document is in standards mode, then the page scroll is on HTML element: document.documentElement.scrollLeft, in quirks mode it’s on the BODY: document.body.scrollLeft.
